Audit Pipeline Overview

Each grievance passes through a structured AI pipeline before being logged in the audit trail.

  1. 01

    Data Ingestion

    Text tickets, PDFs, audio files, and images are registered in Cloud Storage and linked via BigQuery ObjectRef.

    • Cloud Storage
    • BigQuery ObjectRef
    • Multi-language Audio
  2. 02

    AI Structuring

    AI.GENERATE_TABLE extracts issue type, urgency level, applicant details, and sentiment from raw multimodal inputs.

    • AI.GENERATE_TABLE
    • Gemini Multimodal
    • NLP Extraction
  3. 03

    Vector Search & Deduplication

    Embeddings are generated and stored; Vector Search surfaces similar past cases and recommended resolutions for auditors.

    • Vector Search
    • Embedding API
    • Deduplication
  4. 04

    Audit Logging & Dashboard

    Structured outputs are persisted in BigQuery audit tables and visualised in Looker Studio for governance teams.

    • BigQuery Tables
    • Looker Studio
    • Scheduled Queries
